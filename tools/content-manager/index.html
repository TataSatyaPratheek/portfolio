Instructions

How to use this tool:

  1. Select the type of content you want to create.
  2. Fill out the form with your content details.
  3. Preview how your content will look on the website.
  4. Generate the JSON and copy it or download the file.
  5. Add the generated JSON to the appropriate file in your portfolio's data folder.

File Locations:

  • Projects: data/projects.json
  • Blog Posts: data/blog.json
  • Experience: data/experience.json
  • Skills: data/skills.json
Important: This tool runs entirely in your browser and does not save any data to a server. Make sure to copy or download your JSON output before closing the page.
